# ¿Que es Angular?
Angular es un framework de desarrollo web creado por Google, diseñado para construir aplicaciones web modernas, rápidas y escalables. Utiliza **TypeScript**, un lenguaje basado en JavaScript con tipado y herramientas avanzadas que mejoran la calidad del código.

## Características clave
### Framework
Angular incluye todo lo necesario para construir aplicaciones complejas:
* Componentes
* Ruteo
* Formularios
* Servicios
* Manejo de HTTP
* Pruebas

### Arquitectura basada en componentes
Una aplicación Angular se construye dividiéndola en componentes: piezas pequeñas y reutilizables que combinan HTML, CSS, y lógica TypeScript.

### TypeScript
Angular está escrito en TypeScript y fomenta su uso para escribir aplicaciones más seguras, organizadas y fáciles de mantener.

### Reactividad y rendimiento
Angular utiliza **RxJS**, un sistema de programación reactiva basado en observables que facilita manejar datos asíncronos y eventos.

### Angular CLI
Incluye una poderosa herramienta de línea de comandos para crear proyectos, generar código y optimizar aplicaciones.

### Desarrollo escalable
Diseñado para aplicaciones pequeñas, medianas y empresariales. Permite modularizar, reutilizar componentes y aplicar buenas prácticas de arquitectura.

### Comunidad y soporte oficial
Al ser desarrollado por Google y tener una gran comunidad, Angular recibe soporte constante, actualizaciones y mejoras.
